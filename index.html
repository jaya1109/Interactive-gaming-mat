🎮 Interactive Gaming Mat

Using ESP32 & Unity Interface

Developed by Aditi

📌 Project Overview

This project presents an interactive gaming system where physical movements are translated into digital game actions using ESP32 and Unity. Pressure sensors embedded in the mat detect foot movements and send signals to the ESP32 microcontroller. The ESP32 processes these signals and communicates with the Unity game, enabling real-time interaction.

🎯 Objectives

  • To develop an interactive gaming system using real-world inputs
  • To integrate ESP32 hardware with Unity software
  • To detect user movement using pressure sensors
  • To enable real-time communication
  • To create an immersive gaming experience

🧩 Components Used

  • ESP32 Microcontroller
  • Pressure Sensors / Velostat Mat
  • Unity Game Engine
  • USB Communication
  • Power Supply

⚙️ Working Process

  • User steps on the mat
  • Sensors detect pressure
  • ESP32 processes signals
  • Data sent to Unity
  • Game reacts instantly
